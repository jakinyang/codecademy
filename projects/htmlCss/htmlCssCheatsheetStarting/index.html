A brief overview

Overview

Elements

Element Description
Content area The real content of the element: text, video, image, etc. It is bound by the content edge
Padding area The area that surrounds the content area up to the border. It is bound by the padding edge.
Border area The area that surrounds the padding and is bound by the border edge.
Margin area The area that surrounds the border and serves as a buffer between the element and other elements
